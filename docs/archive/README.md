# Archive

Historical documentation preserved for reference. These files are no longer the authoritative source — see the main docs for current guidance.

## What's Here

- **phases/** — Phase prompts and completion summaries (Phases 0–9). Useful for understanding how the lab evolved and the reasoning behind each phase's decisions.
- **implementation-plan.md** — The original phased roadmap written at Phase 0A. Superseded by the phase checkboxes in README.md and current docs.
- **repository-structure.md** — Early snapshot of the directory layout. Superseded by README.md's repository structure tree.
- **superpowers.md** — Pre-skills guide to the Superpowers pattern. Content migrated to [skills/README.md](../../skills/README.md) and [guides/write-a-skill.md](../guides/write-a-skill.md).
- **operator-rules.md** — Original monolithic operator rules document. Split into focused docs under [admin/](../admin/). Preserved here intact.

## When to Look Here

- Investigating why a decision was made during a specific phase
- Understanding the evolution of the lab's safety model or harness design
- Tracing the origin of a policy or convention
- Historical context for an incident post-mortem
